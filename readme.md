# BLIP Image Captioner

Interactive **Streamlit** app using **BLIP** to generate image captions in real time.  
Upload an image, and the app generates a descriptive caption instantly.

## Features
- Upload JPG/PNG images
- Real-time image captioning using BLIP
- Clean, centered Streamlit layout
- Easy-to-run Python code

## How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt
streamlit run app.py
